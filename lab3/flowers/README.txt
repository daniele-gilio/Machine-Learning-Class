Flowers data set
================


This is a subset of the `Oxford Flower Datasets' by Maria-Elena
Nilsback and Andrew Zisserman.

https://www.robots.ox.ac.uk/~vgg/data/flowers/

This subset contains 80 images for each of 12 species of flowers.  For
each class 60 images are in the training set and 20 form the test set.

All the images have been resized to 224x224 pixels.

The `train_augmented' subdirectory contains the training set with data
augmentation, with 18 variations for each training image.
